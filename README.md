# Project 1: HR Analytics Dashboard

## Business Problem
The HR department wants to analyze employee attrition and workforce trends to identify departments with high turnover and improve retention strategies.

## Dataset
HR employee dataset containing employee details, department, salary, tenure, and attrition status.

## Tools Used
- Power BI
- DAX
- Power Query

## Key Metrics
- Attrition Rate
- Employee Count
- Average Tenure
- Department-wise Attrition

## Key Insights
- Attrition rate was highest in the Sales department.
- Employees with lower tenure showed higher turnover.
- Overtime employees had a higher probability of leaving.

# Project 2: Customer Profitability Analysis

## Business Problem
Businesses need to understand which regions, customers, and products generate the highest profit to improve decision-making.

## Dataset
Sales dataset containing order details, revenue, cost, and regional sales information.

## Tools Used
- Power BI
- DAX
- Data Modeling

## Key Metrics
- Total Revenue
- Profit Margin
- Regional Performance
- Customer Profitability

## Key Insights
- Certain regions generated high revenue but low profit margins.
- A small group of customers contributed a large percentage of total revenue.
- Profitability varied significantly across product categories.
